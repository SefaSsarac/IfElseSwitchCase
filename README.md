 bu program kullanıcının meyve almak istediği bir durumda hangi meyvenin seçildiğine bağlı olarak meyvenin fiyatını ekrana yazdıran bir konsol uygulamasıdır. Kullanıcıya meyve adı sorulur ve girilen meyve adına göre program, if ve switch karar yapıları kullanarak ilgili meyvenin fiyatını belirler. Eğer kullanıcının girdiği meyve listede yoksa, program "Diğer bütün meyveler için fiyat: 4 TL" mesajını gösterir.
